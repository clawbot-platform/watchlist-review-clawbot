# Watchlist Review Clawbot

A Clawbot platform domain worker for reviewing sanctions and watchlist alerts generated by existing screening systems.

This repository is designed as a **review assistant**, not a screening engine and not an entity-matching engine. It helps compliance teams reduce false positives by normalizing alert data, extracting deterministic evidence, retrieving grounded policy context, calling shared identity services, and producing conservative, human-reviewable recommendations.

## Why this repo exists

High-volume watchlist screening environments often produce large numbers of false positives. Existing screening engines remain the system of record, but analysts still spend significant effort reviewing alert files and case payloads.

Watchlist Review Clawbot exists to make that review step more structured, explainable, and scalable while preserving human review and conservative escalation.

## Platform role

This repo is a **domain worker** in the broader Clawbot platform.

### Owned here
- alert normalization
- canonical alert contracts
- deterministic feature extraction
- retrieval request construction
- review prompt construction
- recommendation packaging
- analyst note generation
- evaluation harness

### Owned elsewhere
- `clawbot-server` → control plane, runs, cycles, artifacts, reviewer workflow, governance
- `claw-identity` → compare and OFAC screening
- `clawmem` → scoped continuity and memory
- `thinkpad-p50` → data/RAG/memory node
- `ai-precision` → Granite model inference
